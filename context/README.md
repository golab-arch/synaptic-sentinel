# Context - Documentos Directores

Este directorio contiene los documentos de contexto del proyecto **SENTINEL**.

## Proposito

Aqui se almacenan todos los documentos que SYNAPTIC usa para:
- Generar sugerencias de proximos pasos (Tab Roadmap)
- Entender el contexto y requisitos del proyecto
- Extraer tareas pendientes y prioridades

## Tipos de Documentos Recomendados

| Tipo | Descripcion | Ejemplo |
|------|-------------|---------|
| **Requisitos** | Especificaciones del cliente o producto | `requisitos_v1.md` |
| **Arquitectura** | Decisiones de diseno tecnico | `arquitectura_sistema.md` |
| **Guias** | Estandares y convenciones del proyecto | `guia_estilos.md` |
| **Notas** | Informacion de reuniones o decisiones | `notas_kickoff.md` |
| **Backlog** | Lista de features o tareas pendientes | `backlog.md` |

## Uso

1. Crea documentos `.md` en esta carpeta
2. SYNAPTIC los analizara automaticamente
3. Las sugerencias apareceran en el Tab Roadmap
